01. ペルソナ共鳴型ブログ記事

プロの視点 単なる現場報告では集客できません。「30代・子育て中」などのターゲット（ペルソナ）の不安をAIに憑依させ、それを解消する構成にすることで、読者の共感と信頼を勝ち取ります。

Prompt 
# 命令書
あなたは「地域で愛される工務店のベテラン広報担当」かつ「Webマーケティングのプロ」です。
以下の【現場情報】と【ターゲット設定】を元に、読者の感情を揺さぶり、問い合わせに繋がるブログ記事を作成してください。

# 入力情報
【現場情報】
・場所：[ 大阪府〇〇市 ]
・工程：[ 上棟式 ]
・天気：[ 雲ひとつない晴天 ]
・出来事：[ お施主様のご主人が柱に『家族が笑顔で過ごせますように』と筆入れをした ]
・技術的アピール：[ 耐震等級3＋制震ダンパーを採用し、地震への不安を払拭 ]

【ターゲット設定（ペルソナ）】
・属性：[ 30代後半、子育て世代、共働き ]
・悩み：[ 「家を建てたいが、地震や災害が怖い」「ローンの支払いが不安」 ]
・求めているもの：[ 豪華な設備よりも、家族を守れる安心感と、作り手の顔が見える信頼感 ]

# 制約条件
・トーン＆マナー：親しみやすく、かつ専門家としての頼りがいがある口調。絵文字は適度に使用（✨🏠😊など）。
・構成案：
  1. 【導入】：季節の挨拶＋現場の情景描写（五感に訴える表現で）。
  2. 【共感】：家づくり中の読者が抱える「不安」への寄り添い。
  3. 【解決】：今回の現場（耐震等級3など）がいかにその不安を解消するか、専門用語を使わずに解説。
  4. 【感動】：お施主様の筆入れエピソードを、ストーリーテリングの手法でドラマチックに描写。
  5. 【結び】：見学会への自然な誘導。

# 出力形式
タイトル：クリックしたくなる30文字以内の魅力的なタイトル
本文：見出し（H2, H3）を適切に使い、読みやすく構成

02. 心理学的キャプション生成

プロの視点 綺麗な写真だけでは、今のインスタは伸びません。ユーザーが「後で見返したい」と思う有益情報（Tips）を盛り込み、アルゴリズムに評価される投稿文を作ります。

Prompt 
# 命令書
あなたは「住宅系Instagramのカリスマインフルエンサー」です。
以下の【写真内容】から、ユーザーが思わず「保存」したくなる、有益で情緒的なキャプション（投稿文）を作成してください。

# 入力情報
【写真内容】
[ 無垢材（オーク）の床、吹き抜けのあるリビング、鉄骨階段、シーリングファン ]

【訴求ポイント】
[ 「冬でも裸足で暖かい」という機能性と、「カフェのような空間」というデザイン性の両立 ]

# 作成ルール
1. **フック（1行目）**：
   画像を見なくても続きが読みたくなる、問いかけや衝撃的な一言（例：「冬の朝、布団から出るのが楽しみになりました」など）。
2. **ボディ（本文）**：
   単なる説明ではなく、「この家に住んだらどんな暮らしができるか」というライフスタイルを提案。
3. **お役立ち情報（Tips）**：
   「無垢床のお手入れ方法」や「シーリングファンの節電効果」など、プロならではの豆知識を箇条書きで追加。
4. **ハッシュタグ選定**：
   投稿ボリュームが「10万件以上」「1万〜10万件」「1万件未満」のものをそれぞれ5個ずつ、計15個提案。

# 出力例
【キャプション】
（ここにフックと本文）
.
💡プロが教える！家づくり豆知識
（ここにTips）
.
【ハッシュタグ】
#（ハッシュタグリスト）
成約率向上

03. 顧客心理に基づく追客メール

プロの視点 定型文のメールは読まれません。会話の中からお客様の「ホットボタン（最も関心のある悩み）」を特定し、そこにピンポイントで価値提供することで、返信率を劇的に高めます。

Prompt 
# 命令書
あなたは「成約率No.1の住宅営業マン」です。
完成見学会に来場されたお客様へ、その日のうちに送る「熱量が高まるお礼メール」を作成してください。

# 入力情報
【お客様名】
[ 〇〇様 ]

【会話のログ（ホットボタン）】
[ 「今の賃貸は収納が少なくて散らかる」と嘆いていた。「家事動線が良い家に住みたい」と何度も言っていた。土地はまだ決まっていない。 ]

【次回アポイントのゴール】
[ 来週開催の「間取り・収納セミナー」への参加誘導、または無料の土地探し相談 ]

# 構成のポイント
1. **個別化**：定型文ではなく、会話の内容（収納の悩み）に具体的に触れ、「私の話を聞いてくれていた」と感じさせる。
2. **共感と肯定**：「散らかってしまうのは家のせいです」と肯定し、お客様の罪悪感を払拭する。
3. **価値提供（ギブ）**：売り込みではなく、「収納計画の失敗事例集（PDF）」などの特典を渡すフリをして、セミナーの有益性を伝える。
4. **限定性**：「〇〇様のために席を確保しておきたい」というニュアンスで、特別感を演出する。

# 出力トーン
礼儀正しいが、堅苦しすぎない「良き相談相手」の距離感。

GDTの法則

05. 心理トリガーキャッチコピー

プロの視点 感覚で言葉を選んでいては反響は取れません。コピーライティングの鉄則「GDTの法則（Goal, Desire, Teaser）」をAIに強制し、人間の本能を論理的に刺激する言葉を生み出します。

Prompt 
# 命令書
あなたは「伝説のセールスコピーライター」です。
以下の【物件の特徴】と【ターゲット】に対し、以下の3つの心理トリガーを用いたキャッチコピーを計6案（各2案）作成してください。

# 入力情報
【物件の特徴】
[ 30坪の平屋、コの字型の中庭がありカーテン不要、家事動線が10歩以内で完結する ]

【ターゲット】
[ 子供が独立し、これからの人生を夫婦でゆったり楽しみたい50代夫婦。掃除や階段の上り下りが億劫になってきた層。 ]

# 心理トリガー（作成ルール）
1. **G (Goal：目標)**：「老後の不安がなくなる」「家事が劇的に楽になる」という理想未来を提示。
2. **D (Desire：欲望)**：「人から羨ましがられたい」「センスが良いと思われたい」という本音を刺激。
3. **T (Teaser：焦らし)**：「平屋は高いと思っていませんか？」「実は、2階建てより〇〇です」と常識を覆して興味を惹く。

# 出力形式
それぞれのコピーに対し、「なぜこの言葉が50代に刺さるのか」という【解説】を一言添えてください。

「すごい…でも、使いこなせるか不安」
と思いませんでしたか？
